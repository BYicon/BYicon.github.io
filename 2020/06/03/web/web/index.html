# 前端工程化历史：

  1. javascrip 诞生到 ajax 出现前 ： 前后端不分离，服务端渲染，js 仅用于处理页面交互；

  2. ajax 出现后：前后端分离；

  3. v8 引擎发布后：意味着 js 可以脱离浏览器；

  4. 之后发布 nodejs，npm，webpack 前端模块化；

  5. react vue 发布，前端进入了 mvvm + 模块化时代。

# 前端工程化解决了什么问题？

  1. 解决全局命名冲突； 模块化；

  2. 编码规范； eslint；

  3. 使用最新 es 规范，babel；

  4. 资源合并压缩。

总结： 提升前端开发效率，开发规范，访问性能等。
